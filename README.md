# Fullstack online shop

Fullstack online shop built with vanilla JavaScript, EJS, NodeJS + Express, Bcrypt (for hashing the passwords), express-session (middleware for handling sessions and cookies), csurf (for protecting against CSRF attacks), connect-mongodb-session and MongoDB. Free to copy, modify and use.
